Hogy használjuk a Gitet?

Elsősorban egyértelműen le kell töltenünk, majd feltelepíteni (Ide linkeltem az oldalt).

Ha telepítettük a Gitet, regisztrálunk a Githubra (Ide linkeltem az oldalt).

Ha regisztáltunk, kezdhetjük is! Célszerű minden projektet külön mappába csinálni. Először regisztrálni kell a Github fiókunkat a Git Bashba (csak egyszer kell megcsinálni), ezt ezekkel a parancsokkal lehet megtenni:

  • git config --global user.name "Github-on megadott név"
  • git config --global user.email "Github-on regisztrált emailcím"

Hozzunk létre a lokális mappánkba egy repositoryt.

  • mkdir repository-name
  • cd repository-name
  • git init

Távoli kapcsolat beállítása a későbbi feltöltéshez

  • új origin nevű távoli kapcsolat beállítása:
    git remote add origin repository-link (github-ból code-ból elérhető a közvetlen link)
  • távoli kapcsolatok lekérdezése:
    git remote -v
  • origin nevű távoli kapcsolat frissítése:
    git fetch origin
  • minden branch lekérdezése:
    git branch –a

Adjuk hozzá a fájlokat az átmeneti tárolóhoz

  • egy fájl hozzáadása:
    git add fájl-név
  • A mappában lévő összes fájl hozzáadása:
    git add .

Státusz lekérése

  • git status

Fájl feltöltése a távoli tárolóba

  • git push origin master

És kész is vagyunk. A fájlaink már a githubos repository-n vannak.
